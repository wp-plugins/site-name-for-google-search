Include Your Site Name in Google Search Results
== Description ==

Use structured data markup on your public website to indicate the preferred name you want Google to display in Search results. You can also provide more than one possible name for your site, and let Google Search algorithms choose between them. Once Google has crawled and indexed the updated page, we can use the provided name in Search results. This plugin adds the proper mark-up to the header of your website.
